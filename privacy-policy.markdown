# Privacy Policy

Juniper Divination Pin Exporter - Last updated: April 3, 2026

## Overview

The Juniper Divination Pin Exporter ("the App") is a personal tool operated solely by Colleen Beach for the purpose of exporting and archiving her own Pinterest content. This policy describes what data the App accesses and how it is handled.

## Data Accessed

The App connects to the Pinterest API using an access token that belongs to the account owner. It reads the following data from that account:

- Pin titles, descriptions, and destination link URLs
- Pin image URLs
- Board names associated with those pins

No data is accessed from any Pinterest account other than the account owner's own account.

## How Data Is Used

All retrieved data is written to a local CSV file on the account owner's personal computer for the sole purpose of personal backup and offline reference. The data is not processed, analyzed, sold, or used for any commercial purpose.

## Data Sharing

The App does not share, transmit, or disclose any data to third parties. No data is uploaded to external servers, databases, or cloud services by this App.

## Data Retention

The App does not retain any data itself. The output CSV file is stored locally and is managed entirely by the account owner.

## Cookies and Tracking

The App does not use cookies, analytics, advertising trackers, or any similar technologies.

## Third-Party Services

The App uses the Pinterest API v5. Pinterest's own privacy policy governs data held within Pinterest's platform and can be found at [policy.pinterest.com](https://policy.pinterest.com/en/privacy-policy).

## Contact

If you have questions about this privacy policy, you may contact the app operator at [arabella@juniperdivination.com](mailto:arabella@juniperdivination.com).
